Implementation

Utilizing data set captured from buoys, this tool shows a storm path and locations of data collection buoys.

Additional functionality: This map would be interactive, data collection points would be colored to match current weather conditions. Conditions update in real time. Slider functionality will allow you to view historical data. Hovering over map points will show data collected at different timepoints.
